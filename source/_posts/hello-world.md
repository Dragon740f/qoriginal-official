## 服务器内

### 1.第一次登录
如果您第一次登录，那么您一定会被要求注册。不要惊慌，使用
```bash
/reg (密码) (重复一次)
```
来登录，在下一次登陆时，您只需要输入
```bash
/login (您的密码)
```
即可登录


More info: [Writing](https://hexo.io/docs/writing.html)
